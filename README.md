# Share-Paper
This is an initiative taken by GDG VIT so that students can share their CATS,TEE question papers so that other students can view it.
    
## Framework : 
    > SailsJS. (Sailsjs v - 0.12.4, Node : v6.1.0, npm : 3.8.6)
    
## DB :
    > Mongo
